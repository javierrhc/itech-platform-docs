# Estándar de Testing

**Estado:** Draft
**Versión:** v0.1.0
**Owner:** Gobernanza ITech
**Dominio:** governance
**Última actualización:** 2026-02-08

---

## Propósito

Definir reglas mínimas para validar calidad y estabilidad de cambios en la
Plataforma ITech.

## Alcance

Aplica a cambios en código, configuración y documentación con impacto operativo.

## Reglas

- Todo cambio funcional debe tener evidencia de validación.
- Las pruebas críticas deben poder ejecutarse de forma repetible.
- Los resultados de pruebas deben ser registrables y trazables.
- Los defectos detectados deben registrarse y priorizarse.

## Siguiente paso
